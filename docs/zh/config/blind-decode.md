# 解析暗水印

## url

- **Type:** `string`
- **Default:** `''`

解码暗水印图片路径。

## mode

- **Type:** `string`
- **Default:** `'canvas'`
- **available values**: `'canvas'`

解码模式。

## fillColor

- **Type:** `string`
- **Default:** `'#000'`

解码 fill color.

## compositeOperation

- **Type:** `string`
- **Default:** `'color-burn'`

解码 composite operation.
