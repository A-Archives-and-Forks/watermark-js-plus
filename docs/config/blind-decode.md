# Blind Watermark Decode

## url

- **Type:** `string`
- **Default:** `''`

Decoding picture path.

## mode

- **Type:** `string`
- **Default:** `'canvas'`
- **available values**: `'canvas'`

Mode of decoding.

## fillColor

- **Type:** `string`
- **Default:** `'#000'`

fill color.

## compositeOperation

- **Type:** `string`
- **Default:** `'color-burn'`

composite operation.
