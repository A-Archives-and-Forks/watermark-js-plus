基础配置项 #

width #

  • Type: number
  • Default: 300

单个水印的宽度。

height #

  • Type: number
  • Default: 300

单个水印的高度。

rotate #

  • Type: number
  • Default: 45

水印旋转角度

以文本的中心为原点旋转。

contentType #

  • Type: string
  • Default: 'text'
  • available values: 'text' | 'image' | 'multi-line-text' | 'rich-text'

水印内容类型。

content #

  • Type: string
  • Default: 'hello watermark-js-plus'

水印内容。

imageWidth #

  • Type: number
  • Default: 0

水印图像宽度。

imageHeight #

  • Type: number
  • Default: 0

水印图像高度。

lineHeight #

  • Type: number
  • Default: 30

水印内容行高。

zIndex #

  • Type: number
  • Default: 10000

z-index

backgroundPosition #

  • Type: string
  • Default: '0 0, 0 0'

background-position

fontSize #

  • Type: number
  • Default: 20

水印内容的字体大小。

fontFamily #

  • Type: string
  • Default: 'sans-serif'

水印内容的字体。

textAlign #

  • Type: string
  • Default: 'center'
  • available values: 'center' | 'left' | 'right'

水印内容水平对齐方式。

textBaseline #

  • Type: string
  • Default: 'middle'
  • available values: 'top' | 'bottom' | 'middle'

水印内容基准线。

fontColor #

  • Type: string
  • Default: '#000'

水印内容字体颜色。

globalAlpha #

  • Type: number
  • Default: 0.5

水印的透明度。

fontWeight #

  • Type: string
  • Default: 'normal'

水印内容字体权重。

mode #

  • Type: string
  • Default: 'default'
  • available values: 'default' | 'blind'

水印模式。

mutationObserve #

  • Type: boolean
  • Default: true

启用侦听水印dom更改。

unique #

  • Type: boolean
  • Default: true

不能重复添加水印。

parent #

  • Type: Element | string
  • Default: 'body'

水印的容器。

onSuccess #

  • Type: Function
  • Default: () => {}

水印添加成功的回调事件。

onBeforeDestroy #

  • Type: Function
  • Default: () => {}

水印删除前的回调事件。

onDestroyed #

  • Type: Function
  • Default: () => {}

水印删除后的回调事件。
